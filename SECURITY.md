# Security

## Public credentials

A Supabase project URL and publishable key may be used in browser code when Row Level Security is enabled.

## Never expose

- Database passwords
- Supabase secret keys
- Supabase service-role keys
- User access tokens
- Private-submission file URLs

## Reporting

Security concerns should be reported privately to the project owner rather than opened as a public issue.

## Living relatives

Public output should avoid full birth dates, addresses, phone numbers, email addresses, workplaces, and other sensitive identifying information unless explicit consent exists.
